# JS Refresher

Welcome to the **JS Refresher** workshop. This repository is your playground for regaining and solidifying your modern Javascript skills.

## Getting Started

To view the Hello World page, simply open `index.html` in any modern web browser.

### Features
- ✨ Modern, sleek design with glassmorphism
- 🎨 Responsive layout with custom Google Fonts
- ⚡ Subtle micro-animations for interactivity
- 🚀 Clean and semantic HTML structure
